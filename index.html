Features & Plans

A breakdown of what each Chronos Guard plan offers.

Basic Plan

This is the ideal starting point for beginners or for small games. It is simple, direct, and easy to set up.

Detections:

  • AntiSpeed
  • AntiJump
  • AntiFly
  • AntiTeleport

Advantages:

  • Simplicity: The scripts are independent, which makes them very easy to implement.
  • Low Server Load: It consumes few resources as the verifications are basic.
  • Easy to Understand: The logic of each script is simple and does not require advanced knowledge.

Disadvantages:

  • False Positives: It is prone to banning users due to errors or bugs, as it lacks an advanced verification system.
  • No Monitoring: You won't receive alerts when a player is banned. You'll have to check the server logs for detections.

Plus Plan

This plan is a big leap in quality. It offers more solid protection and better control, ideal for games looking to grow.

Detections:

  • Includes all from the Basic Plan.
  • AntiNoClip
  • IntegrityCheck (script injection detection).

Advantages:

  • Modularity: It uses a central script (AntiCheatCore) that handles all sanctions and notifications, making it easy to add new detections.
  • Discord Monitoring: It sends detection notifications to a Discord channel, allowing you to stay on top of what's happening in your game.
  • Flexible Sanctions: You can configure sanctions for players to be kicked, temporarily banned, or permanently banned.

Disadvantages:

  • No Scoring System: There can still be false positives as players are sanctioned for each individual violation, rather than for a behavioral pattern.
  • No Manual Control: You cannot manually kick or ban players from within the game.

Premium Plan

This is the most complete and professional solution, with an advanced architecture that simulates a cloud system. It's ideal for games with a large player base that need robust and precise protection.

Detections:

  • Includes all from the Plus Plan.
  • AntiFlood (autoclicker/macro detection).
  • Behavioral Analysis: The system analyzes movement and behavior patterns to detect more sophisticated cheats.

Advantages:

  • Cloud Simulation: The "cloud" processes detections in the background, which reduces your server's CPU and memory load.
  • Minimizes False Positives: The "suspicion score" system only applies a sanction when it is sure the player is cheating, after a prolonged analysis.
  • Total Control: You have an in-game administration interface to manually sanction players, giving you complete control over your community.
  • Detailed Information: Discord notifications are more detailed, including the UserID and the reason for the sanction.

Disadvantages:

  • Higher Complexity: The initial setup is more complex due to the "cloud" logic and the administration GUI.
  • Dependency: The system relies on the AntiCheatCore to function, so an error in that script could affect the entire system.

FAQ

Frequently asked questions about Chronos Guard.

1. General

Is this system compatible with my game?

Our system works with all Roblox games, but players on consoles (Xbox) will not be affected by the anti-cheat.

Why is this anti-cheat different from others?

Unlike other systems, ours prioritizes mitigations over direct punishments. It also uses a background analysis system to avoid false positives.

2. Configuration

Do I need a special account to use the system?

No. You only need a **Discord account** to receive detection notifications.

How do I configure the system in my game?

To use the Plus or Premium plan, you need to enable HTTP requests in your game and paste your Discord webhook URL into the AntiCheatCore script.

3. Detections and Penalties

How does it detect cheats?

The basic system uses simple movement checks (speed, height). The Premium plan uses a suspicion points system that accumulates evidence before penalizing, making it more accurate.

Why wasn't a player banned immediately?

To avoid false positives, the Premium system only applies a penalty when it is certain that a player is cheating. Detections that don't require immediate intervention are processed in the "cloud" for a longer, more precise analysis.

What is "Script Injection" detection?

It detects if a player has inserted external scripts in their character's GUI. Our Premium system checks this to prevent bans for legitimate Roblox scripts.

Documentation

A deeper look into Chronos Guard's features.

Functions and Permissions

Each plan offers a different level of control and functionality. The system handles penalties automatically, although in more advanced plans, administrators can intervene.

  • Basic Plan: Detections result in an automatic kick or permanent ban.
  • Plus Plan: Penalties are more flexible, as you can configure if the sanction is a temporary or permanent ban.
  • Premium Plan: Administrators can manually manage players using an in-game GUI. They can also mute players for minor offenses.

The Cloud (Simulation)

Our system doesn't use a real cloud, but we have replicated its advantages.

  • Low Server Impact: The system processes detections that don't require immediate intervention in the background. This reduces your server's CPU and memory load.
  • Reliable Detections: The "cloud" offers greater integrity and stability. The system analyzes client behavior for a longer period to detect patterns more accurately.
  • No Interruptions: The delay in detection does not interrupt the game, as the cloud only applies a penalty when the system is certain that the player is cheating.

Updates

Our system is designed to stay one step ahead of cheaters with continuous improvements to detection algorithms. Updates are applied regularly to ensure your game is always protected.

Webhooks Integration

  • Plus Plan: You will receive basic notifications about player bans.
  • Premium Plan: Notifications are more detailed. Discord messages include the UserID and the specific reason for the penalty, helping you identify cheaters more accurately.
